Pour lancer l'application
java -jar rpg_SB.java depuis la commande MS-DOS

Créer un héro
Si le nom donner est une chaine vide, l'application créer un héro par default.
sinon cela créé votre héro. (surcharge du constructeur)

Ensuite on peut
-> afficher votre héro (rédéfinition de l'affichage entre la class Personnage et Hero)
-> afficher un objet de votre héro (association entre le class Hero et Item)
-> quitter (en quittant affichage du toString de l'objet theHero)

il faut aller sur https://www.draw.io/
